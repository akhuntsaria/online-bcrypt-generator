## What is this?
Yet another online bcrypt generator.

## How is it different from others?
You can select rounds AND bcrypt version (even if there's no practical difference).

## Requirements
* Node >= v12 (Tested on v16.3.0)

## Getting started
```shell script
npm install --also=dev
npm run serve
```
